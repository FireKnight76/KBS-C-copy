## Database importeren

1. open XAMPP
2. start Apache en MySQL

![image](https://github.com/FireKnight76/KBS-C-copy/blob/main/assets/159452984/332903814-e28bb266-e912-4423-9308-13909c4c75a8)

4. klik op een van de verbindingen (of maak een nieuwe aan indien nodig)
